Weather Project

Here's a brief description of the project and of the used tools. What is the purpose of this assignement? The purpose of it is to produce a JAVA Application to get the weather of a city (or of an address) chosen by the user. To do this I used Google's Geocoding Service API to obtain the city's geographic coordinates; then I sent them to OpenWeatherMap to get the city's current weather. Those are the mainly steps of building my application: 1) Create Location class which contains location info (with methods to create the two URLs) 2) Create Weather class which contains weather info and location 3) Create Location Request class that makes Http request and creates a XML file with the answer 4) Create Location Coordinates Draw that draws Coordinates from the XML file using XPath 5) Create Weather Request class that makes Http request using coordinates and creates a XML file with the answer 6) Create Weather Coordinates Draw that draws the answer from the XML file using XPath 7) Classes test 8) Graphic User Interface
